Parallel Social, Inc

A cross-platform desktop app built with JavasScript, leveraging technologies like Firebase, Google Cloud, and AWS to run completely serverlessly. Features of Parallel include: messaging, voice calls, video calls, group chats or servers, synced music playback, Spotify playlist import, and many more. Music playback occurs locally with a custom music-fetching algorithm and music discovery uses the Apple Music API. The desktop application is signed and notarized by Apple. Premium subscriptions are handled using the Stripe API and grants access to additional features. A legal C-corporation entity was formed in Delaware for this business, which was later sold. The app currently runs on Linux, Windows, and MacOS.
