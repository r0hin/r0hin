NoteBar Documentation

Overview

I built this application to bridge the gap between great design and security in a note taking application. I built NoteBar with these things in mind, hence why everything you do within NoteBar stays on your computer. It is saved in your localStorage.

Code

I support users contributing to my project, everything is possible on our github repo. There, you can also submit bugs, or feature requests.

Support

To give you the best experience possible when using this app, we have a dedicated software discord server. You can join it here.

Installation

Installing the application is actually extremely easy. Open the main website with Google Chrome, then choose the download button, open the downloaded file and get started using it!
