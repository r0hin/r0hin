Polar Bears

A chemistry-based choose-your-own adventure game! This website hosts V1 of this project. This project is completely free for personal and educational use. In this game, you will choose your own adventure in a story involving questions on the following topics:

  • Structures and Properties of Matter
  • Organic Chemistry
  • Energy Changes and Rates of Reactions
  • Chemical Systems and Equilibrium
  • Electrochemistry

The storylines of this game were carefully crafted with the help of OpenAI's Generative Pre-trained Transformer (GPT-3) 3 which is a textual machine learning model for private, non-commerical, personal use. In other words, elements of this story were not generated by a human!


Start Game


How to play: To play, simply select the button above. You will be given details about the story and you will be asked to choose an option to continue your story. After each choice, you must answer a chemistry question. If you answer correctly, you can try again but one point will be added to your score (less points is better, like golf). You can always play again to improve your score!

Question Content: The questions that will be asked are based on high-school-level chemistry. You can view the complete question/answer directory here. You can also view the full story mockup here. Finally, you can see the source code for this website here.
