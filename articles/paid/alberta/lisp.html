Lisp


Lisp is a characteristic of wrong tooth positioning, worn upper front teeth or exaggerated thickness of the palate. Keep in mind - lisp is normal with new dentures - a period of adjustment is necessary for the tongue to get used to the new set of dentures. However there are certain remedies that can be tried when designing a new set if the problem does not go away with time.
